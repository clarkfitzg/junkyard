We explore some properties of the multinormal distribution.

library(MASS)
library(ggplot2)

n = 5000
mu = c(0, 0)

showmv = function(sigma, center = mu){
    # Plot a multivariate normal with covariance matrix sigma
    s1 = mvrnorm(n, center, sigma)
    k1 = kde2d(s1[, 1], s1[, 2])
    rng = c(-3, 3)
    image(k1, xlim=rng, ylim=rng)
    contour(k1, add = TRUE)
}

Observe that we get ellipses, but no rotation happens when the covariances are zero.

Now let's try with nonzero covariance. Using this matrix causes the points to be collinear, lying on the line y = x.

This becomes less intuitive, but basically we've stretched out the variance in the x direction and correlated it with y.

rotate = function(s, theta){
    # Rotate by theta radians
    r = matrix(c(cos(theta), sin(theta), -sin(theta), cos(theta)), nrow=2)
    r %*% s
}

# 45 degree rotation
